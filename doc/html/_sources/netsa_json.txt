:mod:`netsa.json` --- JSON Wrapper Module
=========================================

.. automodule:: netsa.json

    The :mod:`netsa.json` module provides a wrapper module for either
    the Python standard library :mod:`json` module, if it is
    available, or an included copy of the `simplejson`_ module,
    otherwise.  Please see the standard library documentation for
    details.

    .. _`simplejson`: http://code.google.com/p/simplejson/
